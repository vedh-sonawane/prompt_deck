PromptDeck — AI Presentation Generator

Transform any idea into a stunning, professional presentation in seconds using the power of AI.

🚀 Overview
PromptDeck is a full-stack AI-powered presentation generator that creates beautiful, professional Google Slides decks from a single text prompt. Simply describe what you want, and PromptDeck handles everything — from generating the content and choosing the perfect color theme, to fetching real images and building the slides directly in your Google account.
Whether you're creating a business pitch deck, a history lesson, a science report, or a quarterly review, PromptDeck produces polished, visually rich presentations that rival professionally designed templates.

✨ Features

🤖 AI-Powered Everything

Content, themes, fonts, layouts, and image queries are all decided by Groq's LLaMA 3.3 70B model
No hardcoded templates — every presentation is unique
AI picks topic-appropriate fonts (e.g. Times New Roman for history, Orbitron for tech)
AI selects color palettes that match the mood and subject of your presentation

🎨 Dynamic Theming

Unlimited theme possibilities — colors and styles are generated fresh for every prompt
Background colors, accent colors, title colors, and body colors all chosen by AI
Style categories: modern, elegant, bold, minimal, playful, corporate

🖼️ Real Images on Every Slide

Pexels API integration fetches high-quality, relevant photos for every single slide
Image queries are generated by AI based on each slide's specific content
Images are composited with overlays for maximum readability

📊 Rich Slide Layouts
Six distinct layout types used throughout every presentation:

Title — Full-bleed hero image with overlay and large centered title
Content — Split layout with text on left, image on right
Quote — Full-bleed background with centered pull quote
Statistic — Large stat number in circle with supporting context
Section — Divider slide to separate major topic areas
Closing — Elegant thank you / summary slide

🔢 Smart Slide Count

Just say "make a 30 slide presentation" and PromptDeck will make exactly 30 slides
If you don't specify, AI decides the appropriate length based on topic complexity
Supports 3 to 60 slides

🔐 Authentication

Google OAuth sign-in via NextAuth.js
Glassmorphism login/signup page with animated starry night backgrounds
Background scene changes every time the page loads
Onboarding flow for new users to personalize their experience

📁 Google Slides Integration

Presentations are created directly in your Google account
Opens automatically in Google Slides after generation
Full editing access — customize further after generation

💅 Beautiful UI

Sleek sidebar navigation
Cursive/handwriting font for homepage greeting
Animated loading screen with step-by-step progress
Framer Motion animations throughout
Fully responsive design

⚙️ Setup & Installation
Prerequisites

Node.js 18+
Python 3.11+
A Google Cloud project with Slides API and OAuth enabled
A Groq API key (free at console.groq.com)
A Pexels API key (free at pexels.com/api)

1. Clone the Repository
bashgit clone https://github.com/yourusername/PromptDeck.git
cd PromptDeck
2. Install Frontend Dependencies
bashnpm install
3. Install Backend Dependencies
bashcd backend
python -m venv .venv
.venv\Scripts\activate  # Windows
pip install fastapi uvicorn groq python-dotenv google-api-python-client google-auth-oauthlib requests
4. Configure Environment Variables
Create backend/.env:
envGROQ_API_KEY=your_groq_api_key
PEXELS_API_KEY=your_pexels_api_key
Create .env.local in the root:
envNEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your_random_secret
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
5. Set Up Google Cloud

Go to console.cloud.google.com
Create a new project
Enable the Google Slides API and Google Drive API
Create OAuth 2.0 credentials (Web application)
Add authorized redirect URIs:

http://localhost:3000/api/auth/callback/google


Download credentials.json and place it in the backend/ folder
Run the auth flow once to generate token.json:

bashpython auth_manager.py
6. Start the Application
Terminal 1 — Backend:
bashcd backend
$env:GROQ_API_KEY="your_key"      # PowerShell
$env:PEXELS_API_KEY="your_key"
uvicorn main:app --reload --port 8000
Terminal 2 — Frontend:
bashnpm run dev
Open http://localhost:3000

🎯 Usage

Go to http://localhost:3000
Type a prompt like:

"Make a 20 slide presentation about the Pacific Scandal"
"Create a 10 slide pitch deck for a food delivery startup"
"Build a science presentation about black holes"


If not signed in, you'll be redirected to the login page
Sign in with Google
New users complete a short onboarding questionnaire
Your presentation is generated and opens automatically in Google Slides

🚧 Known Limitations

Google OAuth redirect URIs can take up to a few hours to propagate after changes
Pexels images must be publicly accessible URLs for Google Slides API to embed them
Groq's LLaMA model may occasionally return malformed JSON which is handled gracefully
Email/password authentication is UI-only and not yet connected to a database
